This simple editor includes templates for quick insertion onto this page, such as calendars, resumes, cover letters, etc.—at the moment, but eventually, I will add additional ones. You can easily edit these templates by clicking below the line that says "EDITABLE CONTENT BELOW" (this area). To perform other actions, click the buttons in the floating menu above. Below, I will explain each part of the floating menu and the save/load buttons area.
  
I will begin with the first button located in the top left corner.    
 
✍️This editor auto-saves on key presses, so if you load a template and want to store it in local storage, click somewhere and hit a key! 
 
'Clear': Clears everything in the editable content area.

'Select All': Select everything! This is good if a person wants to select all text and change font style, size, etc.

'Copy All': Copies all the content. The content will remain formatted, so it is great for creating templates for large email content.

'Reload,' 'Undo,' 'Redo': These all work nearly the same as most editors, except the reload.

'CF': Clear Formatting! Sometimes, a line below or some border repeats when working with a heavily formatted or styled template. This should stop that from happening!

The following two drop-downs contain several common font styles and standard font sizes. The next is to change the text color. (left box/button), select the text, and use the one beside it (right box/button) to highlight the text.

Heading Special

The 'HS' (Heading Special) button attempts to mimic the current special heading for the templates or inserts a default large heading.

Highlight background ← This is the 'HL' button.

 'Link': Highlight a link or text in this area, and it will create a clickable link that opens the web address. The save as PDF converts to img, so this feature will not work in that format.

The 'Img' button will add images. If saving this as an edited HTML file for use on a webpage, the image must be hosted, or the files must be in the exact location or custom edited in a source code editor. However, keeping it as a Word or a PDF will save the image to that file format. You can also drag the image around in this area. Once the image is inserted, it will be small. Click once to enlarge it, and click again to make it medium. After the three sizes are cycled, you must backspace or undo the image and insert it again.

'Emojis' - click the desired emoji to be inserted into the selected area. 

'Line Spacing' ← This feature will make the content more readable and make something look better.

'Print': The prompt will explain additional details when clicking this button. 

The 'Calendar Tool' is still in progress but is usable. It can be saved and loaded in this editor to do more styling.

 

§ Common Shortcut Keys (just some additional tips and info)

Emdash — hold alt and press 0151 —this adds emphasis to a sentence or a change in thought that feels like it should be in the same sentence or included.

→ alt+26, ← alt+27, © alt+0169, ™ alt+0153, § alt+21, ¶ alt+0182

'Windows Key' plus a period opens the Windows Emojis Tool. Although this editor has some commonly used emojis, it is also a neat tool.
